# 📖 Project Overview

**Borobhai.com** is a modern web-based platform designed to connect university alumni and current students through a centralized networking system.

The primary goals of this project are to:

* Help students connect with senior alumni
* Provide career guidance
* Share internship and job opportunities
* Build professional networking
* Maintain alumni communication with the university

---

# ⚙️ Project Development Approach

The project will be developed using:

* Approximately **40% custom human-written code**
* Approximately **60% AI-assisted/generated code** *(or more if necessary)*

> All generated code will be manually implemented, customized, tested, and validated during development.

The developer will mainly focus on:

* Implementing features
* Integrating modules
* Customizing functionality
* Testing and debugging with AI assistance
